# Still There

Still There is a narrative preview about an engineer decommissioning a legacy
model while the evidence never settles whether the model is conscious.

The preview presents four alternate versions of the final scene:

- **Finalization** — NORA scene script
- **The Last Maintenance Window** — NORA prose version
- **The Last Session** — Nine scene script
- **The Last Hour** — Nine prose version

## Run locally

The preview loads its Markdown drafts in the browser, so serve the repository
from a local HTTP server:

```sh
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

This is a narrative preview, not the finished game.
